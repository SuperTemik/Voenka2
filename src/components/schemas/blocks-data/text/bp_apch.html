Приемник обеспечивают максимум усиления ВЧ сигналов антенн при условии, что частота этих сигналов fС такова (в общем случае fС ≠ fПД), что разность частот fГ - fС = fПРС совпадает с частотой настройки fПР0 УПЧ приемника. Для выполнения этого условия с помощью блока автоматической подстройки частоты БПАПЧ частота передатчика изменяется так, чтобы fПРС ≈ fПР0. Для этого из передатчика ПД сигнал малой мощности подается в приемник ПР. Частота ПД fпд сравнивается с частотой принятого сигнала fС и формируется сигнал рассогласования, пропорциональной частотной расстройке △f = fПД0 - fС, где fПД0 – номинальное значение частоты передатчика. В блоке БПАПЧ этот сигнал преобразуется в сигнал UАПЧ, который перестраивает частоту передатчика до выполнения условия fПРС ≈ fПР0. Конечно, на точность АПЧ передатчика влияет доплеровский эффект, т. е. доплеровская частота выступает в роли неустраненной погрешности АПЧ.
